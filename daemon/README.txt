Dependencies
============

Before you run Orbited, you will need:

    * Python 2.5
    * Twisted 8.1.0
    * cjson or simplejson


Quick Start
===========

Install the dependencies, and start orbited with:

    python start.py

Try the Chat demo at:

    http://localhost:8001/static/demos/chat/

NB: This will connect to the IRC channel #orbited at
    irc.freenode.net:6667. For it to work, your computer needs to be
    able to open an TCP connection to that server.

You can also see the other demos at:

    http://localhost:8001/static/demos/

To properly install and configure orbited, read the documentation at:

    http://orbited.org/wiki/Documentation
